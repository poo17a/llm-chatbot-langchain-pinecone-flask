# llm-chatbot-langchain-pinecone-flask
End-to-end Retrieval-Augmented Generation (RAG) chatbot using LLMs, LangChain, Pinecone vector database, and Flask. Supports semantic search, real-time chat, and knowledge-based responses with custom data.

# How to run?
### Steps:

Clone the repository

```bash
git clone https://github.com/poo17a/llm-chatbot-langchain-pinecone-flask.git
```

### Step 01 - Create a conda environment after opening the repository

```bash
Conda create -n medibot python=3.10 -y
```

```bash
Conda activate medibot
```

### Step 02: Install the requirements
```bash
pip install -r requirements.txt
```
